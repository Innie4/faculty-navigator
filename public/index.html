What this isn’t

This isn’t indoor navigation. GPS works outdoors with 3–10 m accuracy. You’ll need to grant location permission when the app asks. The map data was gathered by hand and may have gaps — if something’s wrong, it can be fixed. This is a student project, not a commercial product.
